# FINAL FANTASY XIV 繁體中文版狩獵工具

## 📋 項目概述

這是一個為FFXIV玩家設計的狩獵怪物管理工具，支持快速搜尋、分類追蹤、批量管理等功能。通過Firebase Firestore雲端儲存，實現實時數據同步與多裝置訪問。

## ✨ 核心功能

### 🎯 狩獵搜尋板
- **全局搜尋** - 按名稱即時查找怪物
- **版本篩選** - 支援 2.0～7.0 所有主線版本
- **地圖篩選** - 依據版本自動載入對應地圖清單
- **等級與FATE篩選** - 快速定位特定等級或FATE怪物
- **討伐筆記職業篩選** ⭐
  - 點擊按鈕激活（亮藍色）
  - 「全部」模式：顯示所有有職業標籤的怪物
  - 「特定職業」模式：只顯示該職業的怪物
  - 支援12個基礎職業與01~50的編號後綴
- **座標複製** - 一鍵複製怪物名稱+地圖+座標到剪貼簿
- **展開/摺疊** - 支援批量切換座標可見性

### 📌 追蹤看板
- **自訂分組** - 建立多個分類分組（預設「常用區域」）
- **拖曳管理** - 滑鼠拖曳怪物卡片到不同分組
- **實時篩選** - 獨立的版本、地圖搜尋與職業篩選
- **批量展開** - 全部展開/摺疊追蹤怪物座標
- **快速移除** - Hover時顯示刪除按鈕

### 🛠️ 管理控制台
- **新增怪物** - 單筆編輯模式
  - 設定名稱、版本、等級、FATE狀態、討伐筆記職業
  - 多座標管理（支援多張地圖）
  - 批次座標解析（自動識別 X:數字 Y:數字 格式）
  
- **批量新增列表** ⭐ 新功能
  - 一次新增多隻怪物
  - 統一設定版本、地圖、等級、FATE、討伐筆記職業
  - 實時預覽與標籤顯示
  - 預設職業設為「劍術士01」（可隨時修改）
  - 一鍵寫入 Firestore
  
- **表格管理** - 檢視所有怪物清單
  - 批量選擇
  - 快速編輯/刪除
  - 多欄位過濾

## 📦 數據結構

### 怪物 (Monster) 物件
```javascript
{
  id: string,              // 唯一識別碼 (m_timestamp_random)
  name: string,            // 怪物名稱
  version: string,         // 版本 (2.0-7.0)
  rank: string,           // 等級 (None, B, A, S, SS)
  isFate: boolean,        // 是否為FATE怪物
  job: string | null,     // 討伐筆記職業標籤 (如 "劍術士01"，null表示無)
  locations: [
    {
      map: string,        // 地圖名稱
      x: string,          // X座標
      y: string           // Y座標
    }
  ]
}
```

### PIN (用戶追蹤) 物件
```javascript
{
  id: string,                // 怪物ID
  groupId: string,          // 所屬分組ID
  updatedAt: number         // 最後更新時間戳
}
```

### 分組 (Group) 物件
```javascript
{
  id: string,               // 分組ID
  name: string,            // 分組名稱
  order: number,           // 排序
  canDelete: boolean       // 是否可刪除
}
```

## 🚀 快速開始

### 使用方式

1. **開啟應用** - 在瀏覽器中打開 `ff14-hunt-finder-v17.html`
2. **登入** - 輸入帳號與密碼進入
3. **選擇板面**
   - 「狩獵板」- 搜尋與追蹤怪物
   - 「管理員」- 編輯/新增/刪除怪物

### 狩獵板操作

```
1. 在搜尋欄輸入怪物名稱
2. 使用篩選器縮小範圍（版本、地圖、等級、FATE、討伐筆記職業）
3. 職業篩選說明：
   - 點擊「討伐筆記」按鈕激活（按鈕轉藍色）
   - 下拉選單預設為「全部」，顯示所有有職業標籤的怪物
   - 選擇特定職業，只顯示該職業的怪物
   - 再次點擊「討伐筆記」按鈕取消篩選
4. 點擊卡片展開座標
5. 點擊座標複製到剪貼簿
6. 點擊Pin圖示加入追蹤看板
```

### 追蹤看板操作

```
1. 使用獨立篩選器搜尋與過濾
2. 滑鼠拖曳怪物卡片移動到不同分組
3. 在卡片Hover時點擊X移除追蹤
4. 點擊分組名稱編輯分組名稱
5. 可新增自訂分組 (+按鈕)
```

### 管理控制台操作

#### 新增單筆怪物
```
1. 點擊 [新增怪物] 按鈕
2. 填入怪物名稱、選擇版本
3. 選擇等級與FATE狀態
4. 新增座標點位 (支援多座標)
   - 使用「批次座標解析」快速匯入
   - 或手動新增單筆座標
5. 點擊 [儲存] 完成
```

#### 批量新增怪物 ⭐
```
1. 點擊 [批量新增列表] 按鈕
2. 設定共用配置
   - 版本 (自動讀取管理員篩選值)
   - 地圖 (自動讀取管理員篩選值)
   - 等級 (自動讀取管理員篩選值)
   - FATE (自動讀取管理員篩選值)
   - 討伐筆記職業 (點擊按鈕激活，預設為劍術士01)
3. 在「名稱輸入」區域貼入多個名稱 (一行一個)
4. 點擊 [解析名稱 →] 預覽
5. 確認預覽無誤後點擊 [確認批量新增]
6. 系統自動寫入 Firestore

**職業操作**：
- 未設置：按鈕灰色，下拉選單隱藏
- 設置後：按鈕藍色，顯示「劍術士01」
- 修改：點擊下拉選單改變職業或編號
- 清除：再次點擊「討伐筆記」按鈕全部清空
```

#### 編輯現有怪物
```
1. 在表格中找到目標怪物
2. 點擊 [Edit] 按鈕
3. 修改名稱、版本、等級、FATE、討伐筆記職業、座標
   - 職業操作同批量新增邏輯
   - 點擊「討伐筆記」按鈕激活職業選擇
   - 可使用下拉選單修改職業/編號
4. 點擊 [儲存] 完成
5. 關閉對話框時職業狀態自動重置
```

#### 刪除怪物
```
1. 在表格中找到目標怪物
2. 點擊 [Trash] 按鈕
3. 確認刪除提示
```

## 💼 討伐筆記職業分類系統

### 職業列表
支援12個基礎職業，每個職業可搭配01~50的編號後綴：

```
劍術士、格鬥家、斧術師、槍術師、弓箭手、幻術師
咒術師、秘術師、雙劍師、黑渦團、雙蛇黨、恆輝隊
```

**職業標籤格式**：`職業名稱 + 編號`  
範例：「劍術士01」、「格鬥家25」、「弓箭手50」

### 職業篩選邏輯

| 按鈕狀態 | 下拉選單顯示 | 查詢結果 |
|---------|-----------|--------|
| 灰色（未激活） | 隱藏 | 不篩選職業 |
| 藍色（激活全部） | 「全部」(*) | 所有有職業標籤的怪物 |
| 藍色（激活特定） | 「劍術士」等 | 該職業的怪物 |

### 交互流程

**查詢視圖**：
1. 點擊「討伐筆記」按鈕 → 亮藍色，下拉選單打開
2. 預設顯示「全部」→ 查詢所有有職業標籤的怪物
3. 選擇具體職業 → 只查詢該職業的怪物
4. 再次點擊「討伐筆記」 → 取消篩選，回到灰色

**批量新增**：
1. 點擊「討伐筆記」按鈕 → 下拉選單打開
2. 預設職業為「劍術士01」
3. 可在職業下拉選單選擇其他職業
4. 在編號下拉選單選擇01~50
5. 點擊「討伐筆記」再次移除選擇 → 清除職業標籤

**編輯怪物**：
1. 點擊「討伐筆記」按鈕 → 下拉選單打開
2. 預設職業為「劍術士01」
3. 修改職業與編號
4. 點擊「討伐筆記」移除 → 清除職業標籤
5. 關閉對話框時自動重制狀態

## 🔧 開發資訊

### 主要組件
- **App** - 主應用容器
- **VersionTag** - 版本標籤組件
- **RankTag** - 等級標籤組件
- **FateTag** - FATE標籤組件
- **Icon** - Lucide圖示包裝器

### 關鍵狀態管理
- `monsters` - 全局怪物清單
- `userPins` - 用戶追蹤記錄
- `customGroups` - 用戶自訂分組
- `filterJob`, `kbFilterJob`, `adminFilterJob` - 各視圖職業篩選狀態（""無篩選, "*"全部, 職業名稱）
- `showJobFilter`, `showKbJobFilter`, `showAdminJobFilter` - 職業下拉選單顯示控制
- `bulkConfig` - 批量新增配置（包含 jobBase, jobSuffix）
- `editJobBase`, `editJobSuffix` - 編輯模式職業狀態
- `showBulkAdd` - 批量新增彈窗控制
- `editData` - 編輯表單狀態

### 主要函數
- `applyFilter()` - 多條件篩選
- `handleBulkParseNames()` - 解析批量名稱
- `executeBulkAdd()` - 執行批量寫入
- `handleBatchParse()` - 座標批次解析
- `copyToClipboard()` - 複製功能
